# 本地运行

## 环境

- Python 3.11+
- Node.js 20+
- npm
- 可选移动端环境：JDK 17、Android SDK、ADB、Appium 3 和 uiautomator2 driver

## 后端

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

健康检查：`http://127.0.0.1:8000/health`；OpenAPI：`http://127.0.0.1:8000/docs`。

后端默认使用本地 SQLite。数据库、上传文件和日志都是本地运行数据，不得提交。

## 前端

```bash
cd frontend
npm install
npm run dev
```

默认访问 `http://127.0.0.1:5173`。生产构建使用：

```bash
npm test
npm run build
```

## 后端测试

```bash
cd backend
python -m unittest discover -s tests
```

测试若需要 AI 成功结果，应显式 mock 模型响应；业务运行路径不允许通过模板或规则降级伪造 AI 成功。

## 仓库治理

从仓库根目录运行：

```bash
python -m unittest tests.test_repository_governance -v
```

运行目录和清理边界见[运行产物说明](运行产物.md)。
