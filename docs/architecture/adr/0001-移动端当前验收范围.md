# ADR-0001：移动端当前验收范围

**状态：** 已接受
**日期：** 2026-07-15

## 决策

当前移动端正式验收范围仅为 Android：设备识别与占用、App 信息、页面与元素资产、单条移动端 Case、Appium 执行，以及截图、XML 和 Appium 日志证据。

iOS、云真机、多设备并发调度和 AI 视觉定位不进入当前正式验收范围。实验性 iOS 或视觉能力必须保持隔离，不能作为默认业务入口。

## 依据与影响

该结论与 PRD 的 `FR-MOB-001` 至 `FR-MOB-015`、`FR-MOB-011` 和 Non-goals 一致。正式移动端 Case 必须引用元素资产；AI 自愈只产生待人工确认的修复提案，不自动修改正式资产。
