# ADR-0004：RAG 当前范围

**状态：** 已接受
**日期：** 2026-07-15

## 决策

当前 RAG 范围包括：受控候选片段、关键词粗筛、真实模型语义重排、来源与召回记录，以及被拒 AI Case 的脱敏负样本检索。

当前不引入独立向量数据库，也不声明已具备完整向量知识库运营能力。允许先使用关系型存储和内存候选集；当候选片段存在且需要语义重排时，模型配置或调用失败必须显式失败，不允许用规则结果冒充 AI 成功。

## 入库边界

只有人工确认的需求、Case、接口、报告摘要和脱敏反馈可成为可信上下文。检查未通过需求、未确认 AI 输出和未脱敏内容不得进入正式知识范围。向量数据库选型和迁移属于后续独立决策。
