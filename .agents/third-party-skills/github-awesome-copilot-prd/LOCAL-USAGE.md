## What It Provides

This local copy is used as a PRD and product manager reference.

The PRD skill emphasizes:

- discovery before drafting
- measurable success criteria
- concrete acceptance criteria
- explicit non-goals
- AI system evaluation strategy
- technical specifications and risk analysis

The Product Manager Advisor emphasizes:

- user need before feature work
- business context in issues
- measurable outcomes
- issue size and phase labeling
- actionable acceptance criteria

## Best Fit For This Project
