# httpbin 全量接口文档

## 文档说明

本文档整理 `httpbin.org` 公开接口，用于接口自动化平台建档、导入解析、调试和生成接口 case。

参考来源：

- [httpbin Swagger 规范](https://httpbin.org/spec.json)
- [httpbin Status codes 页面](https://httpbin.org/#/Status_codes)

## 基础信息

| 项目 | 内容 |
| --- | --- |
| 服务名称 | httpbin.org |
| 版本 | 0.9.2 |
| 基础地址 | `https://httpbin.org` |
| 协议 | HTTPS |
| 说明 | 一个用于测试 HTTP 请求和响应行为的服务 |

## 详细接口说明

### HTTP Methods

#### GET `/get`

返回请求中的 query 参数。

示例：

```http
GET https://httpbin.org/get?name=test
```

推荐断言：

```json
[
  { "type": "status_code", "expected": 200 },
  { "type": "json_equals", "path": "args.name", "expected": "test" }
]
```

#### POST `/post`

返回请求中的 POST 参数、JSON body、form data、headers 等。

示例：

```http
POST https://httpbin.org/post
Content-Type: application/json

{"name":"test"}
```

#### PUT `/put`

返回请求中的 PUT 参数。

#### PATCH `/patch`

返回请求中的 PATCH 参数。

#### DELETE `/delete`

返回请求中的 DELETE 参数。

### Auth

#### GET `/basic-auth/{user}/{passwd}`

使用 HTTP Basic Auth 鉴权。

| 参数 | 位置 | 类型 | 说明 |
| --- | --- | --- | --- |
| user | path | string | 用户名 |
| passwd | path | string | 密码 |

响应：

| 状态码 | 说明 |
| --- | --- |
| 200 | 鉴权成功 |
| 401 | 鉴权失败 |

示例：

```http
GET https://httpbin.org/basic-auth/demo/123456
Authorization: Basic ZGVtbzoxMjM0NTY=
```

#### GET `/hidden-basic-auth/{user}/{passwd}`

使用 HTTP Basic Auth 鉴权。失败时返回 `404`，而不是普通的 `401`。

#### GET `/bearer`

使用 Bearer Token 鉴权。

示例：

```http
GET https://httpbin.org/bearer
Authorization: Bearer demo-token
```

响应：

| 状态码 | 说明 |
| --- | --- |
| 200 | 鉴权成功 |
| 401 | 鉴权失败 |

#### GET `/digest-auth/{qop}/{user}/{passwd}`

使用 Digest Auth 鉴权。

| 参数 | 位置 | 类型 | 说明 |
| --- | --- | --- | --- |
| qop | path | string | `auth` 或 `auth-int` |
| user | path | string | 用户名 |
| passwd | path | string | 密码 |

#### GET `/digest-auth/{qop}/{user}/{passwd}/{algorithm}`

使用 Digest Auth，并指定算法。

| 参数 | 位置 | 类型 | 说明 |
| --- | --- | --- | --- |
| algorithm | path | string | `MD5`、`SHA-256`、`SHA-512` |

#### GET `/digest-auth/{qop}/{user}/{passwd}/{algorithm}/{stale_after}`

使用 Digest Auth，并指定算法和 stale 过期设置。

### Status codes

#### `/status/{codes}`

返回指定 HTTP 状态码。如果传入多个状态码，服务端会随机返回其中一个。

支持方法：

```text
GET / POST / PUT / PATCH / DELETE / TRACE
```

| 参数 | 位置 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
| codes | path | string | 是 | 状态码；多个状态码用逗号分隔 |

示例：

```http
GET https://httpbin.org/status/200
GET https://httpbin.org/status/404
GET https://httpbin.org/status/200,400,500
```

响应：

| 状态码类型 | 说明 |
| --- | --- |
| 1xx | Informational responses |
| 2xx | Success |
| 3xx | Redirection |
| 4xx | Client Errors |
| 5xx | Server Errors |

### Response inspection

#### GET `/cache`

如果请求中存在 `If-Modified-Since` 或 `If-None-Match`，返回 `304`；否则返回普通 GET 响应。

#### GET `/cache/{value}`

设置 `Cache-Control` 响应头。

| 参数 | 位置 | 类型 | 说明 |
| --- | --- | --- | --- |
| value | path | integer | 缓存秒数 |

#### GET `/etag/{etag}`

假设资源拥有指定 ETag，并根据请求头 `If-None-Match`、`If-Match` 返回对应响应。

| 参数 | 位置 | 类型 | 说明 |
| --- | --- | --- | --- |
| etag | path | string | ETag 值 |

#### GET/POST `/response-headers`

根据 query 参数返回响应头。

示例：

```http
GET https://httpbin.org/response-headers?X-Demo=hello
```

# httpbin 状态码接口文档

## 接口名称

返回指定 HTTP 状态码

## 接口说明

该接口用于测试客户端对不同 HTTP 状态码的处理能力。调用方可以在路径中指定一个或多个状态码，服务端会返回对应的 HTTP 状态。

如果传入多个状态码，httpbin 会从中随机返回一个。

参考页面：[httpbin Status codes](https://httpbin.org/#/Status_codes)

## 基础地址

```text
https://httpbin.org
```

## 接口地址

```text
/status/{codes}
```

## 请求方式

```text
GET
```

## 路径参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| codes | string | 是 | 一个或多个 HTTP 状态码。多个状态码用逗号分隔 | `200`、`404`、`200,400,500` |

## 请求示例

### 返回 200

```http
GET https://httpbin.org/status/200
```

### 返回 404

```http
GET https://httpbin.org/status/404
```

### 随机返回多个状态码之一

```http
GET https://httpbin.org/status/200,400,500
```

## 响应说明

该接口通常不返回业务 JSON body，重点是 HTTP 响应状态码。

| 场景 | 请求路径 | 预期状态码 |
| --- | --- | --- |
| 成功状态 | `/status/200` | `200` |
| 未授权 | `/status/401` | `401` |
| 禁止访问 | `/status/403` | `403` |
| 资源不存在 | `/status/404` | `404` |
| 服务端错误 | `/status/500` | `500` |
| 多状态码随机 | `/status/200,400,500` | `200` / `400` / `500` 之一 |
